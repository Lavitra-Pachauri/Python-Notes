A Python Interpreter is a software that reads your Python program line by line and executes it immediately. Unlike compiled languages, Python does not convert the entire program into machine code before running.

👥 Aliasing and Cloning of List Objects +

Aliasing

Aliasing means two variables refer to the same list object. Changes made through one variable are reflected in the other.

list1=[10,20,30] list2=list1 list2.append(40) print(list1) print(list2)

Output

[10,20,30,40] [10,20,30,40]

Cloning

Cloning creates a separate copy of the original list. Changes made to one list do not affect the other.

list1=[10,20,30] list2=list1.copy() list2.append(40) print(list1) print(list2)

Output

[10,20,30] [10,20,30,40]
Aliasing Cloning
Same Memory Different Memory
Changes Affect Both Changes Affect Only Copy
⭐ Memory Trick Aliasing = Same Object 👥 Cloning = New Copy 📄

📦 Tuple Packing and Unpacking +

Tuple Packing means storing multiple values into a single tuple. Python automatically packs the values into one tuple object.

Tuple Unpacking means assigning tuple elements into separate variables. The number of variables must be equal to the number of tuple elements.

Rahul 18 92.5

A Dictionary is a collection of data stored as key : value pairs. Each key must be unique and is used to access its corresponding value. Dictionary objects are enclosed within { } braces.
