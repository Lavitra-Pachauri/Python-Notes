🐍 Python Introduction +

What is Python?

Python is a high-level, interpreted programming language that is designed to be simple, readable, and powerful. It allows programmers to write fewer lines of code while solving complex problems.

Why is Python Popular?

  • Easy syntax
  • Beginner friendly
  • Used in AI
  • Web Development
  • Automation
  • Data Science
print("Hello Gargi")
💡 Trick: Think of Python as English language for computers.
🌍 Python in Real World +

Python is used by companies around the world for building websites, machine learning applications, automation tools, scientific research, cybersecurity, and cloud computing.

  • Artificial Intelligence
  • Automation
  • Desktop Software
  • Games
  • Web Applications
  • Data Analysis
Exam Tip: Remember at least five real-world uses of Python.
⭐ Why Python? +

Python is one of the easiest programming languages to learn. It has simple syntax, a huge collection of libraries, and works on Windows, Linux, and macOS.

  • Easy to Learn

  • Powerful Libraries

💻 Python Interpreter +

What is a Python Interpreter?

A Python Interpreter is a software that reads your Python program line by line and executes it immediately. Unlike compiled languages, Python does not convert the entire program into machine code before running.

How it Works?

  1. Read the code.
  2. Check for errors.
  3. Execute line by line.
  4. Display the output.
💡 Trick:
Interpreter = Read → Understand → Execute

🎯 Conditional Statements +

Conditional statements help a program make decisions. Python provides

  • if
  • if else
  • if elif else
Decision Making = Conditional Statements
✅ if Statement +

The if statement executes a block only when the condition is True.

age=18 if age>=18: print("Eligible")

Practice Program

num=10 if num>0: print("Positive Number")
If condition is False, Python skips the block.
🔀 if else Statement +
marks=45 if marks>=33: print("Pass") else: print("Fail")

Program

num=7 if num%2==0: print("Even") else: print("Odd")
Only one block executes.
🚀 if elif else +

Used when multiple conditions are checked.

marks=82 if marks>=90: print("A") elif marks>=75: print("B") elif marks>=60: print("C") else: print("D")
elif means "Else If".
🔢 range() Function +

What is range()?

The range() function generates a sequence of numbers. It is commonly used with loops.

Syntax

range(start, stop, step)

Examples

for i in range(5): print(i) Output 0 1 2 3 4

Another Example

for i in range(2,11,2): print(i) Output 2 4 6 8 10
💡 Trick Start Included ✔ Stop Excluded ❌
🔁 while Loop +

The while loop executes as long as the condition is True.

i=1 while i<=5: print(i) i=i+1

Output

1 2 3 4 5
Always update the loop variable to avoid an infinite loop.
🔄 for Loop +

The for loop is used to repeat a block of code for each item in a sequence.

for i in range(1,6): print(i)

Output

1 2 3 4 5
Use for when you know the number of iterations.
⛔ break Statement +

The break statement immediately stops the loop.

for i in range(10): if i==5: break print(i) Output 0 1 2 3 4
break = Exit the loop immediately.
⏭ continue Statement +

The continue statement skips the current iteration and moves to the next iteration.

for i in range(6): if i==3: continue print(i) Output 0 1 2 4 5
continue = Skip only one iteration.
⏸ pass Statement +

The pass statement does nothing. It is used as a placeholder.

for i in range(5): pass print("Loop Finished")
pass = Do Nothing.
🎯 else Statement with Loop +

Python allows an else block with loops. The else block executes when the loop finishes normally.

for i in range(5): print(i) else: print("Loop Completed")
If break executes, the else block will not run.
🛡 assert Statement +

The assert statement checks whether a condition is True. If the condition is False, Python raises an AssertionError.

age=18 assert age>=18 print("Eligible")

Example 2

num=-5 assert num>0
💡 Exam Tip assert is mainly used during debugging and testing.

💻 Program 1 : split() Method in Python +

📖 Explanation

The split() method is used to divide a string into multiple parts. It returns the separated values as a list. By default it splits using spaces, but we can also specify our own separator.

💻 Python Code

str1 = "Be a Creator"
print(str1.split())

str2 = "banana, mango, apple"
print(str2.split(', '))

str3 = "1:2:3:4:5:6:7:8:9"
print(str3.split(':'))

str4 = "hello world!"
print(str4.split('d'))

🖥 Output

['Be', 'a', 'Creator']
['banana', 'mango', 'apple']
['1', '2', '3', '4', '5', '6', '7', '8', '9']
['hello worl', '!']
⭐ Exam Tip: split() always returns a list.
💻 Program 2 : split() using maxsplit +

📖 Explanation

The maxsplit parameter tells Python how many times the string should be split. After the specified number of splits, the remaining text stays together.

💻 Python Code

nums = '1, 2, 3, 4, 5, 6, 7, 8, 9'

print(nums.split(', ',2))
print(nums.split(', ',1))
print(nums.split(', ',5))
print(nums.split(', ',0))

🖥 Output

['1', '2', '3, 4, 5, 6, 7, 8, 9']
['1', '2, 3, 4, 5, 6, 7, 8, 9']
['1', '2', '3', '4', '5', '6, 7, 8, 9']
['1, 2, 3, 4, 5, 6, 7, 8, 9']
⭐ Exam Tip: maxsplit controls the maximum number of splits.

1 : banana
2 : mango

💻 Program 12 : reversed() Function +

📖 Explanation

The reversed() function returns a reverse iterator. Convert it to a list using list() to display the reversed values.

💻 Python Code

# For String
str = "bca"
print(list(reversed(str)))

# For Tuple
tpl = ("m","a","n","g","o")
print(list(reversed(tpl)))

# For Range
rng = range(1,6)
print(list(reversed(rng)))

# For List
lst = [1,3,5,7,9]
print(list(reversed(lst)))

🖥 Output

['a', 'c', 'b']

['o', 'g', 'n', 'a', 'm']

[5, 4, 3, 2, 1]

[9, 7, 5, 3, 1]
⭐ reverse() modifies the original list.
⭐ reversed() returns an iterator and does not change the original object.
